# 🎭 Demo Features для презентации

## 🎨 Визуальные эффекты

### 1. Динамический фон
- **Анимированные частицы** с подключениями между ними
- **Интерактивность мыши** - частицы следуют за курсором
- **Адаптивная плотность** - количество частиц зависит от размера экрана

### 2. Переходы между слайдами
- **Fade + Scale эффекты** для плавного появления слайдов
- **Staggered анимации** - элементы появляются поочередно
- **Cubic-bezier кривые** для естественных движений

### 3. Типографика
- **Градиентные заголовки** с анимацией цвета
- **Анимированные подчеркивания** под h2
- **Floating эффекты** для главного заголовка

## 🎯 Интерактивные элементы

### 1. Карточки проблем и решений
- **Hover эффекты** с подсветкой и трансформацией
- **Градиентные overlay** при наведении
- **Smooth shadow transitions**

### 2. Use Cases иконки
- **Микро-анимации** при hover
- **Scale + rotate эффекты**
- **Drop-shadow с акцентным цветом**

### 3. CTA секция
- **Анимированный градиентный фон**
- **Pulsing border effects**
- **Enhanced glass morphism**

### 4. QR-код
- **Настоящий QR для @Urlexx**
- **Анимированная рамка** при hover
- **3D transform эффекты**

## ⌨️ Управление

### Клавиши
- `→` или `Space` - следующий слайд
- `←` - предыдущий слайд
- Мышь - перемещение частиц

### Навигация
- **Animated dots** с backdrop blur
- **Smooth scroll** между секциями
- **Auto-detection** текущего слайда

## 🚀 Производительность

### Оптимизации
- **GPU-accelerated animations** (transform, opacity)
- **RequestAnimationFrame** для частиц
- **Will-change** для активных элементов
- **Debounced scroll listeners**

### Совместимость
- ✅ Chrome/Edge 80+
- ✅ Firefox 75+
- ✅ Safari 13+
- ✅ Мобильные браузеры

# 🎤 Руководство по проведению презентации

## 🎯 Перед началом

### Техническая подготовка
1. Откройте https://e2-e5.github.io/ai-first-enhanced-presentation/ в Chrome/Edge
2. Переключитесь в **полноэкранный режим** (F11)
3. Отключите уведомления и закройте лишние программы
4. Убедитесь, что мышь работает плавно (для эффектов частиц)

### Проверочный список
- [ ] Интернет соединение стабильно
- [ ] Звук отключен или настроен
- [ ] Яркость экрана/проектора оптимальна для темной темы
- [ ] Есть запасной вариант (локальная копия)

## 🎭 Сценарий презентации (7 минут)

### Слайд 1: Заголовок (30 сек)
> "Добро пожаловать! Я хочу показать вам AI First - копилот для руководителя."

**Действия:**
- Медленно проведите мышью по экрану - покажите частицы
- Дайте аудитории полюбоваться анимацией заголовка
- Не спешите переходить дальше

### Слайд 2: Проблема (1.5 минуты)
> "Начнем с проблемы, которую все знают..."

**Действия:**
- Стрелка → для перехода
- Медленно наведите мышь на каждую карточку проблемы
- Обратите внимание на подсветку и анимации
- Зачитайте ключевые пункты

**Ключевые сообщения:**
- Данные разбросаны по системам
- Рутина съедает время руководителя  
- Сложно получить целостную картину

### Слайд 3: Решение (1.5 минуты)
> "А вот как мы это решаем..."

**Действия:**
- Transition эффект должен впечатлить
- Hover по карточкам решения (голубое свечение)
- Подчеркните контраст с проблемами

**Ключевые сообщения:**
- Единый мозг системы
- Мгновенный анализ
- Полная автоматизация рутины

### Слайд 4: Как работает (1 минута)
> "Принцип работы простой..."

**Действия:**
- Покажите красивую анимацию центрального блока
- Обратите внимание на floating эффект
- Кратко, без технических деталей

### Слайд 5: Применения (1.5 минуты)
> "9 конкретных направлений применения..."

**Действия:**
- Это самый интерактивный слайд!
- Кликните по каждой иконке (покажите анимацию)
- Выберите 3-4 самых релевантных для аудитории
- Дайте конкретные примеры

**Фокус на:**
- Договоры и документы (для юристов/руководителей)
- Отчеты (для аналитиков)
- Презентации (для всех)

### Слайд 6: Тест-драйв (1 минута)
> "Переходим к самому интересному..."

**Действия:**
- Hover по CTA блоку (градиентный эффект)
- Подчеркните уникальность предложения
- Создайте ощущение эксклюзивности

**Ключевые сообщения:**
- Только 2 компании
- Себестоимость внедрения
- Полный пакет + 3 месяца поддержки

### Слайд 7: Контакты (30 сек)
> "Если заинтересовались - вот мои контакты"

**Действия:**
- Наведите на QR-код (анимированная рамка)
- Покажите, что это рабочий QR для Telegram
- Оставьте слайд открытым для фото

## 🎪 Продвинутые техники

### Управление вниманием
- **Движение мыши** = фокус внимания на элементе
- **Паузы** после анимаций дают время усвоить информацию
- **Ритм** - быстрые переходы для динамики, медленные для важного

### Обработка вопросов
- Используйте навигацию точками для быстрого возврата к слайдам
- Клавиши ← → для навигации во время ответов
- Можете показать hover эффекты повторно для уточнений

### Технические моменты
- Если подвисла анимация - перезагрузите страницу (Ctrl+R)
- Если проектор не показывает эффекты - переключитесь на другой браузер
- В крайнем случае есть backup версия без анимаций
